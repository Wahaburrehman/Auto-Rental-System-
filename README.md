 This project is a front-end based Car Rental Website developed using HTML, CSS, and JavaScript.
The main objective of this project is to design and develop a responsive and interactive car rental platform interface that simulates a real-world online car rental service.

Problem Statement:
Traditional car rental systems often lack a modern, user-friendly online interface.
This project provides a structured and visually appealing solution that allows users to browse available cars, view details, and interact with the platform smoothly.

System Overview:
AutoRent is a static web-based application that allows users to:
•	View available cars with images
•	Browse featured car categories
•	Explore pricing cards
•	Experience an interactive user interface
•	Navigate through different sections smoothly
The system is fully front-end based and runs in a web browser.

Project Structure:
The project consists of the following main files:
•	index.html → Main webpage structure
•	styles.css → Styling and layout design
•	script.js → JavaScript functionality
•	assets/ → Images and graphical content
•	Car images 
•	Logo and banner images

Technologies Used:
•	HTML5 → Page structure
•	CSS3 → Styling and responsive layout
•	JavaScript → Interactivity and dynamic behavior
•	Images & Media Assets → Visual presentation

Features Implemented:
1. Responsive Design
•	Clean and modern UI
•	Organized car display cards
•	Structured sections (Hero, Cars, Pricing, etc.)
2. Interactive Components
•	JavaScript-based UI interactions
•	Navigation handling
•	Dynamic effects for better user experience
3. Car Showcase
•	Multiple car models displayed with images
•	Visual pricing cards
•	Professional layout similar to real rental websites

Design Approach:
The website follows a structured UI layout:
•	Header Section → Logo and navigation
•	Hero Section → Main banner image
•	Car Display Section → Featured vehicles
•	Pricing Cards Section → Rental plans
•	Footer Section → Branding and additional info
The design focuses on clarity, simplicity, and real-world usability.
